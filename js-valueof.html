Apparently you can screw with type coercion in javascript by overriding the valueOf function. Here are some crazy examples: 
var a = { valueOf: function() { return 1; } };
isNaN(a) === false // a is a Number!
Number(a) // returns 1
a == 1; // returns true
a + 3; // return 4
+a; // returns 1
JS is a cool language...
